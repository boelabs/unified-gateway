```json
{ "publicModel": "minimax", "adapterKey": "minimax", "upstreamModel": "MiniMax-M2", "credentials": { "apiKey": "..." } }
```

- **Credentials**: `apiKey` only. No `baseUrl` needed — defaults to `https://api.minimax.io/v1`.
- **Default transport**: Chat Completions.
- Catalog: `src/adapters/minimax/catalog.json`.
